# 🎬 MERN VideoHub — YouTube Clone

A feature-packed **YouTube-style video-sharing platform** built with the **MERN stack** (MongoDB, Express, React, Node.js). This application enables users to securely upload, stream, and manage videos, all via a sleek and responsive interface.

---

## 🚀 Overview

This full-stack app replicates core features of YouTube:

- 🔐 Secure authentication using JWT
- 📤 Upload and manage videos
- 📺 Stream videos directly from the cloud (via Cloudinary)
- 🔍 Search videos by title or description
- 🖥️ Clean UI with mobile-first responsive design

The tech stack includes:

- **MongoDB** for database
- **Express.js** + **Node.js** for the backend
- **React** for the frontend
- **Cloudinary** for cloud video/image storage
- **Tailwind CSS** for styling

---

## 🛠️ Installation Guide

Follow the steps below to run the project locally:

### 1. Clone the Repository

```bash
git clone https://github.com/am7876388/YoutubeClone.git
cd YoutubeClone
```

### 2. Install Dependencies & Run

#### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

#### Backend Setup (in a new terminal)
```bash
cd backend
npm install
npm start
```

### 3. Open in Browser

Visit: [http://localhost:5173](http://localhost:5173)
